Here are the instructions to run our project
1. Run TCPWebserver.py
2. In the browser of your machine, type 'localhost:55555/index.html' to access the main page
3. Click on any one of the links. All links except 'original pic' will send you a png file with a message encoded in it
4. When the picture opens, right-click and select save as
5. save it in the same folder where your client code is kept
6. run Client.py. This file will decode the image and open the message in browser.
